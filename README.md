# Student Productivity Dashboard

## Overview
The Student Productivity Dashboard is a web application that helps students track their study sessions and monitor productivity.

## Problem Statement
Students often do not track how much time they spend on different subjects. This leads to poor time management and lack of productivity analysis.

## Solution
This project provides a simple dashboard where users can:
- Add study sessions
- Store data in a database
- View total study hours
- View subject-wise study hours

## Features
- Add study sessions
- Store records in MySQL database
- View total study hours
- View subject-wise study hours
- Display all sessions in a table

## How to Run
1. Install Python  
2. Install required packages:
   pip install -r requirements.txt  

3. Start MySQL (XAMPP)  
4. Create database using database.sql  
5. Run the app:
   python app.py  

6. Open in browser:
   http://127.0.0.1:5000
